# 客户管理系统

　　《客户管理系统》是专门针对于公司、企业对客户管理而开发的一套软件。软件以客户为中心，通过各种方式记录与客户沟通的详细信息，方便对客户服务的跟踪、更好的发掘潜在客户。主要包括(商机管理、客户管理、行动管理、服务管理)。
